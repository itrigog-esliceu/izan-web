Microsoft Windows

El sistema operatiu Windows és un software desarrollat per sa companyia Microsoft, la seva funcio principal es la de crear un entorn més accesible en el ordinador per la comoditat dels usuaris.

Característiques

Windows a anyadit varies característiques que avui en dia són fonamentals, algunes com:

  • Connexió entre dispositius
  • Ventana d'inici
  • Centre de notificacions
